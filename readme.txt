Zet de ReserveerEenvoudig widget gemakkelijk op uw website.

== Description ==

Deze plugin voegt de ReserveerEenvoudig widget toe aan de front-end van uw website. U hoeft alleen het BedrijfsGUID in te stellen op de instellingenpagina.

== External Service ==

Deze plugin maakt gebruik van een externe service van ReserveerEenvoudig. Het laadt een script van de volgende URL:

- `https://reserveereenvoudig.nl/widget.js`

Bij het gebruik van deze plugin wordt de BedrijfsGUID naar de externe service gestuurd om de widget correct weer te geven.

**Links naar het privacybeleid en de algemene voorwaarden van Taurus Kassasystemen:**

- [Privacybeleid en Algemene Voorwaarden](https://www.tauruskassasystemen.nl/contact/algemene-voorwaarden)

Het is belangrijk dat gebruikers deze voorwaarden lezen en begrijpen hoe hun gegevens worden verwerkt en gebruikt door de externe service.

== Installation ==

1. Upload de plugin bestanden naar de `/wp-content/plugins/reserveereenvoudig/` directory, of installeer de plugin via de WordPress plugin directory.
2. Activeer de plugin via het 'Plugins' menu in WordPress.
3. Ga naar 'Instellingen' > 'ReserveerEenvoudig' om je BedrijfsGUID in te stellen.

== Frequently Asked Questions ==

= Hoe stel ik mijn BedrijfsGUID in? =

Ga naar 'Instellingen' > 'ReserveerEenvoudig' en vul je BedrijfsGUID in.

= Welke gegevens worden er naar de externe service gestuurd? =

De plugin stuurt alleen de BedrijfsGUID naar `https://reserveereenvoudig.nl/` om de widget correct te laden.
